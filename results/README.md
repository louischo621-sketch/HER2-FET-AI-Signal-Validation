# Results

This folder contains the main figures generated from the HER2 FET AI-assisted signal validation workflow.

## Key Figures

- `fig04_representative_waveforms_raw_drop.png`  
  Representative signal waveforms and steady-state concentration responses for different signal-quality categories.

- `fig05a_calibration_linear_scale.png`  
  HER2 calibration curve comparison on a linear concentration scale before and after signal validation.

- `fig05b_calibration_log_scale.png`  
  HER2 calibration curve comparison on a logarithmic concentration scale before and after signal validation.

## Additional Recommended Figures

The following figures may also be included if available:

- `confusion_matrix.png`

- `ai_probability_threshold.png`

## Notes

The figures summarize the main outcomes of the signal validation workflow. The goal of this analysis is not to replace the HER2 calibration curve, but to identify reliable FET sensing channels before calibration curve construction.
